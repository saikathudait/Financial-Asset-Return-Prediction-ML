# Retail Sales Data Analysis Using Python

## Project Overview
This project performs an exploratory data analysis (EDA) on a retail store dataset to uncover sales trends, customer behavior, and product performance. The analysis supports data-driven decision-making for inventory management, marketing strategy, and operational improvement.

## Files Included
- `sales_analysis.py` – Python script containing data cleaning, feature engineering, analysis, and visualizations.
- `Salesdata.xlsx` – Retail sales dataset used for analysis.
- `README.md` – Project documentation.

## Tools and Libraries Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook / Python Script

## Analysis Performed
- Data cleaning and preprocessing
- Feature engineering (Day, Month)
- Descriptive statistics
- Sales analysis by product line, branch, city, and customer type
- Payment method and quantity analysis
- Correlation analysis
- Time-based sales trends

## Key Insights
- Health and Beauty and Electronic Accessories are top-performing product lines
- Members generate higher total sales than normal customers
- Ewallet and Cash are the most preferred payment methods
- Sales performance varies across branches and cities
